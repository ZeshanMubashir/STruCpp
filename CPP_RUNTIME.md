## Overview

The C++ runtime library provides the foundation for all code generated by STruC++. It implements:

- IEC 61131-3 data type wrappers
- Standard library functions
- Standard function blocks
- Variable forcing mechanism
- Utility functions

### Library Structure

```
runtime/
├── include/
│   ├── iec_types.h          # IEC type wrappers
│   ├── iec_std_functions.h  # Standard functions

│   └── iec_utils.h          # Utility functions
```

## Design Goals

### 1. Type Safety

Provide compile-time type checking while maintaining IEC 61131-3 semantics:

```cpp
IEC_INT x = 10;
IEC_REAL y = 3.14;
// x = y;  // Compile error: type mismatch
x = static_cast<IEC_INT>(y);  // Explicit conversion required
```

### 2. Performance

Generate code that is as efficient as hand-written C++:

- Inline everything possible
- Zero-cost abstractions
- No virtual functions in hot paths
- Minimal memory overhead

### 3. Debuggability

Make generated code easy to debug:

- Preserve variable names
- Readable type names
- Clear error messages
- GDB/LLDB friendly

### 4. Variable Forcing

Support OpenPLC's variable forcing mechanism:

- Force any variable to a specific value
- Override normal program logic
- Inspect forced status
- Unforce variables

### 5. Real-Time Capability

Ensure deterministic, real-time execution:

- No dynamic memory allocation
- No exceptions in generated code
- Predictable execution time
- Bounded memory usage

## IEC Type Wrappers

### Base Template

All IEC types are based on a common template:

```cpp
template<typename T>
class IECVar {
public:
    using value_type = T;
    
    // Constructors
    IECVar() : value_{}, forced_{false}, forced_value_{} {}
    
    explicit IECVar(T v) : value_{v}, forced_{false}, forced_value_{} {}
    
    // Copy/move constructors and assignment operators
    IECVar(const IECVar&) = default;
    IECVar(IECVar&&) = default;
    IECVar& operator=(const IECVar&) = default;
    IECVar& operator=(IECVar&&) = default;
    
    // Accessors
    T get() const noexcept {
        return forced_ ? forced_value_ : value_;
    }
    
    void set(T v) noexcept {
        if (!forced_) {
            value_ = v;
        }
        // Note: When forced, set() updates the underlying value
        // but get() still returns the forced value
    }
    
    // Forcing support
    void force(T v) noexcept {
        forced_ = true;
        forced_value_ = v;
    }
    
    void unforce() noexcept {
        forced_ = false;
    }
    
    bool is_forced() const noexcept {
        return forced_;
    }
    
    // Implicit conversion for natural syntax
    operator T() const noexcept {
        return get();
    }
    
    // Assignment from raw value
    IECVar& operator=(T v) noexcept {
        set(v);
        return *this;
    }
    
    // Arithmetic operators (for numeric types)
    IECVar& operator+=(T v) noexcept {
        set(get() + v);
        return *this;
    }
    
    IECVar& operator-=(T v) noexcept {
        set(get() - v);
        return *this;
    }
    
    IECVar& operator*=(T v) noexcept {
        set(get() * v);
        return *this;
    }
    
    IECVar& operator/=(T v) noexcept {
        set(get() / v);
        return *this;
    }
    
    // Increment/decrement
    IECVar& operator++() noexcept {
        set(get() + 1);
        return *this;
    }
    
    IECVar operator++(int) noexcept {
        IECVar tmp = *this;
        ++(*this);
        return tmp;
    }
    
    IECVar& operator--() noexcept {
        set(get() - 1);
        return *this;
    }
    
    IECVar operator--(int) noexcept {
        IECVar tmp = *this;
        --(*this);
        return tmp;
    }
    
private:
    T value_;
    bool forced_;
    T forced_value_;
};

// Binary operators (friend functions for natural syntax)
template<typename T>
inline IECVar<T> operator+(const IECVar<T>& a, const IECVar<T>& b) noexcept {
    return IECVar<T>(a.get() + b.get());
}

template<typename T>
inline IECVar<T> operator-(const IECVar<T>& a, const IECVar<T>& b) noexcept {
    return IECVar<T>(a.get() - b.get());
}

template<typename T>
inline IECVar<T> operator*(const IECVar<T>& a, const IECVar<T>& b) noexcept {
    return IECVar<T>(a.get() * b.get());
}

template<typename T>
inline IECVar<T> operator/(const IECVar<T>& a, const IECVar<T>& b) noexcept {
    return IECVar<T>(a.get() / b.get());
}

// Comparison operators
template<typename T>
inline bool operator==(const IECVar<T>& a, const IECVar<T>& b) noexcept {
    return a.get() == b.get();
}

template<typename T>
inline bool operator!=(const IECVar<T>& a, const IECVar<T>& b) noexcept {
    return a.get() != b.get();
}

template<typename T>
inline bool operator<(const IECVar<T>& a, const IECVar<T>& b) noexcept {
    return a.get() < b.get();
}

template<typename T>
inline bool operator>(const IECVar<T>& a, const IECVar<T>& b) noexcept {
    return a.get() > b.get();
}

template<typename T>
inline bool operator<=(const IECVar<T>& a, const IECVar<T>& b) noexcept {
    return a.get() <= b.get();
}

template<typename T>
inline bool operator>=(const IECVar<T>& a, const IECVar<T>& b) noexcept {
    return a.get() >= b.get();
}
```

Define aliases for all IEC 61131-3 types:

```cpp
// Boolean
using IEC_BOOL = IECVar<bool>;

// Bit strings

using IEC_DWORD = IECVar<uint32_t>;
using IEC_LWORD = IECVar<uint64_t>;

// Signed integers
using IEC_SINT = IECVar<int8_t>;
using IEC_INT = IECVar<int16_t>;
using IEC_DINT = IECVar<int32_t>;
using IEC_LINT = IECVar<int64_t>;

// Unsigned integers
using IEC_USINT = IECVar<uint8_t>;
using IEC_UINT = IECVar<uint16_t>;
using IEC_UDINT = IECVar<uint32_t>;
using IEC_ULINT = IECVar<uint64_t>;

// Real numbers
using IEC_REAL = IECVar<float>;
using IEC_LREAL = IECVar<double>;

// Time types (stored as milliseconds)
using IEC_TIME = IECVar<int64_t>;
using IEC_DATE = IECVar<int64_t>;
using IEC_TIME_OF_DAY = IECVar<int64_t>;
using IEC_DATE_AND_TIME = IECVar<int64_t>;

// String types
class IEC_STRING {
public:
    static constexpr size_t MAX_LENGTH = 255;
    
    IEC_STRING() : length_(0), forced_(false) {
        data_[0] = '\0';
    }
    
    explicit IEC_STRING(const char* str);
    
    const char* get() const noexcept {
        return forced_ ? forced_data_ : data_;
    }
    
    void set(const char* str);
    
    size_t length() const noexcept {
        return forced_ ? forced_length_ : length_;
    }
    
    void force(const char* str);
    void unforce() noexcept { forced_ = false; }
    bool is_forced() const noexcept { return forced_; }
    
    // Operators
    IEC_STRING& operator=(const char* str) {
        set(str);
        return *this;
    }
    
    operator const char*() const noexcept {
        return get();
    }
    
private:
    char data_[MAX_LENGTH + 1];
    size_t length_;
    bool forced_;
    char forced_data_[MAX_LENGTH + 1];
    size_t forced_length_;
};
```

### Array Types

Arrays are implemented as wrappers around C++ arrays:

```cpp
template<typename T, size_t N>
class IECArray {
public:
    using value_type = T;
    using size_type = size_t;
    
    IECArray() = default;
    
    // Element access
    T& operator[](size_type index) {
        return data_[index];
    }
    
    const T& operator[](size_type index) const {
        return data_[index];
    }
    
    // Bounds-checked access
    T& at(size_type index) {
        if (index >= N) {
            // Handle error (in production, this should be a compile-time check)
        }
        return data_[index];
    }
    
    // Size
    constexpr size_type size() const noexcept {
        return N;
    }
    
    // Iterators
    T* begin() noexcept { return data_; }
    const T* begin() const noexcept { return data_; }
    T* end() noexcept { return data_ + N; }
    const T* end() const noexcept { return data_ + N; }
    
private:
    T data_[N];
};

// Example usage
using IntArray10 = IECArray<IEC_INT, 10>;
using RealMatrix3x3 = IECArray<IECArray<IEC_REAL, 3>, 3>;
```

## Standard Function Blocks

### Timer: TON (On-Delay)

```cpp
class TON {
public:
    // Inputs
    IEC_BOOL IN;
    IEC_TIME PT;
    
    // Outputs
    IEC_BOOL Q;
    IEC_TIME ET;
    
    // Constructor
    TON() : IN(false), PT(0), Q(false), ET(0), 
            start_time_(0), running_(false) {}
    
    // Execution method (called each scan cycle)
    void operator()() {
        int64_t current_time = get_current_time_ms();
        
        if (IN.get() && !running_) {
            running_ = true;
            start_time_ = current_time;
        }
        
        if (running_) {
            int64_t elapsed = current_time - start_time_;
            ET = IEC_TIME(elapsed);
            Q = IEC_BOOL(elapsed >= PT.get());
        }
        
        if (!IN.get()) {
            running_ = false;
            Q = IEC_BOOL(false);
            ET = IEC_TIME(0);
        }
    }
    
private:
    int64_t start_time_;
    bool running_;
};
```

### Counter: CTU (Up Counter)

```cpp
class CTU {
public:
    // Inputs
    IEC_BOOL CU;
    IEC_BOOL R;
    IEC_INT PV;
    
    // Outputs
    IEC_BOOL Q;
    IEC_INT CV;
    
    // Constructor
    CTU() : CU(false), R(false), PV(0), Q(false), CV(0),
            last_cu_(false) {}
    
    // Execution method
    void operator()() {
        // Reset
        if (R.get()) {
            CV = IEC_INT(0);
            Q = IEC_BOOL(false);
            last_cu_ = false;
            return;
        }
        
        // Count on rising edge
        bool cu_current = CU.get();
        if (cu_current && !last_cu_) {
            CV = IEC_INT(CV.get() + 1);
        }
        last_cu_ = cu_current;
        
        // Update output
        Q = IEC_BOOL(CV.get() >= PV.get());
    }
    
private:
    bool last_cu_;
};
```

### Edge Detection: R_TRIG (Rising Edge)

```cpp
class R_TRIG {
public:
    // Input
    IEC_BOOL CLK;
    
    // Output
    IEC_BOOL Q;
    
    // Constructor
    R_TRIG() : CLK(false), Q(false), last_clk_(false) {}
    
    // Execution method
    void operator()() {
        bool clk_current = CLK.get();
        Q = IEC_BOOL(clk_current && !last_clk_);
        last_clk_ = clk_current;
    }
    
private:
    bool last_clk_;
};
```

## Memory Management

### Static Allocation

All variables are statically allocated:

```cpp
// Program class
class MyProgram {
public:
    MyProgram() = default;
    
    void operator()() {
        // Program logic
    }
    
private:
    // All variables are members (static allocation)
    IEC_INT counter;
    TON timer;
    IEC_BOOL flag;
};
```

### No Dynamic Allocation

The runtime library avoids dynamic allocation:

- No `new`/`delete`
- No `std::vector`, `std::string`, etc.
- Fixed-size arrays
- Stack or static storage only

This ensures:
- Deterministic execution
- No memory fragmentation
- Real-time capability
- Predictable memory usage

## Variable Forcing

### Forcing Mechanism

Variable forcing allows runtime override of variable values:

```cpp
// Force a variable
my_program.counter.force(42);

// Check if forced
if (my_program.counter.is_forced()) {
    // Variable is forced
}

// Unforce a variable
my_program.counter.unforce();
```

## Performance Considerations

### Inline Everything

All wrapper methods are inline:

```cpp
inline T get() const noexcept { ... }
inline void set(T v) noexcept { ... }
```

This allows the compiler to optimize away the wrapper overhead.

### Zero-Cost Abstraction

When forcing is not used, the wrapper has zero overhead:

```cpp
// Without forcing
IEC_INT x = 10;
IEC_INT y = 20;
IEC_INT z = x + y;

// Compiles to (with optimizations):
int16_t x = 10;
int16_t y = 20;
int16_t z = x + y;
```

### Compile-Time Configuration

Use preprocessor macros to disable features:

```cpp
#ifdef IEC_DISABLE_FORCING
    // Forcing disabled - remove forced_ members
    template<typename T>
    class IECVar {
        T value_;
        // No forced_ or forced_value_
    };
#endif
```

### Optimization Levels

Support different optimization levels:

- **Debug**: Full checks, no optimizations
- **Release**: Optimizations enabled, minimal checks
- **Real-Time**: Maximum optimizations, no checks

## OpenPLC Integration

### Runtime API


### Located Variables

Support located variables (mapped to I/O):

```cpp
// ST: VAR input_button AT %IX0.0 : BOOL; END_VAR

// Generated C++:
class MyProgram {
public:
    void operator()() {
        // Read from I/O
        input_button = read_digital_input(0, 0);
        
        // Program logic
        output_led = input_button;
        
        // Write to I/O
        write_digital_output(0, 0, output_led.get());
    }
    
private:
    IEC_BOOL input_button;  // Mapped to %IX0.0
    IEC_BOOL output_led;    // Mapped to %QX0.0
};
```

### Time Functions

Provide time functions for OpenPLC:

```cpp
// Get current time in milliseconds
int64_t get_current_time_ms();

// Get current date/time
IEC_DATE_AND_TIME get_current_datetime();
```

## Summary

The C++ runtime library provides:

✅ **Type-safe wrappers** for all IEC 61131-3 types
✅ **Complete standard library** (functions and FBs)
✅ **Variable forcing** support
✅ **Zero-cost abstractions** with inline optimization
✅ **Real-time capability** (no dynamic allocation)
✅ **OpenPLC integration** (runtime API, I/O mapping)
✅ **Debugger-friendly** (readable types, preserved names)

This runtime library enables STruC++ to generate efficient, maintainable, and debuggable C++ code that integrates seamlessly with the OpenPLC ecosystem while providing excellent performance for real-time PLC applications.
